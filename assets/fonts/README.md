# Font Assets

This directory should contain Inter font files from Google Fonts.

## Required Font Files

Download Inter font family from [Google Fonts](https://fonts.google.com/specimen/Inter) and place the following files here:

- `Inter-Regular.ttf` (weight: 400)
- `Inter-Medium.ttf` (weight: 500)
- `Inter-SemiBold.ttf` (weight: 600)
- `Inter-Bold.ttf` (weight: 700)

## Quick Download

You can download Inter from:
https://fonts.google.com/specimen/Inter

Or use the direct download link:
https://github.com/rsms/inter/releases

Make sure to extract and place the `.ttf` files in this directory.
